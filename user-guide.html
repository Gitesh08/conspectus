Overview

Conspectus is a powerful Chrome extension that leverages built-in AI to convert descriptions into beautiful diagrams directly in your browser.

Key Features

  • Uses Chrome's built-in AI to understand diagram requirements
  • Generates various diagram types: flowcharts, sequence diagrams, class diagrams, state diagrams, and entity relationship diagrams
  • Simple, intuitive interface with real-time diagram generation
  • Clean, modern design with a responsive side panel interface
  • Zoom controls and diagram download functionality
  • AI-powered diagram description improvement suggestions
  • Diagram summarization feature

Kickstart

Installation

  1. Clone the repository: git clone https://github.com/Gitesh08/conspectus.git
  2. Open Chrome and navigate to chrome://extensions/
  3. Enable "Developer mode" in the top right corner
  4. Click "Load unpacked" and select the extension directory

Setup

Requirements

  • Chrome version 129.0.6639.0 or later (Canary or Dev channel recommended)
  • At least 22 GB of free storage space
  • GPU with 4 GB+ VRAM
  • Windows 10/11, macOS 13 (Ventura) or later, or Linux

Enabling AI Features

  1. Open Chrome and go to chrome://flags
  2. Enable the following flags:
    • #optimization-guide-on-device-model
    • #prompt-api-for-gemini-nano
    • #rewriter-api-for-gemini-nano
    • #summarization-api-for-gemini-nano
  3. Relaunch Chrome
  4. Navigate to chrome://components and refresh the page
  5. Locate the "Optimization Guide On Device Model" component
  6. Click on the "Update" button for this component
  7. Wait for the update to complete (this may take a few minutes)
  8. After a successful update, the status should show as "Up-to-date"

⚠️ If you don't see the "Optimization Guide On Device Model" component, please check this discussion for troubleshooting steps.

API Usage

Conspectus utilizes three main AI APIs provided by Chrome:

1. Prompt API

The Prompt API allows Conspectus to interpret natural language instructions and convert them into diagram specifications.

const session = await window.ai.languageModel.create({
    systemPrompt: getSystemPrompt(diagramType)
});
const mermaidSyntax = await session.prompt(query);

2. Rewriter API

The Rewriter API is used to enhance and improve diagram descriptions provided by users.

const diagramRewriter = await window.ai.rewriter.create({
    context: 'Transform user-provided input into a precise, technical, concise description...'
});
const enhancedDescription = await diagramRewriter.rewrite(userInput);

3. Summarization API

The Summarization API helps condense complex diagram information into concise summaries.

const diagramSummarizer = await window.ai.summarizer.create({
    type: 'key-points',
    length: 'medium',
    context: 'You are an expert at analyzing and summarizing technical diagrams...'
});
const diagramSummary = await diagramSummarizer.summarize(mermaidSyntax);
